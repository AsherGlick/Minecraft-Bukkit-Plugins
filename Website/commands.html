Regions Commands

Claim

/claim <region name>
Use this command to claim a 8x8 plot of land with a given name for $5000. The corners are marked with four torches.

Expand

/expand
/ex
Add another a 8x8 plot of land to an adjacent plot that you own for $1000. The corners are marked with four torches.

Dark Expand

/dark-expand
/darkexpand
/dx
Add another a 8x8 plot of land to an adjacent plot that you own for $1000. No torches are placed. Useful for claiming plots that you have already built in an dont need torches for.

Rename

/rename <new name>
Changes the name of the plot you are standing in for $6000.

Add Owner

/add-owner <player name>
/addowner <player name>
/ao <player name>
Add another player as an owner of your plot. Owners have full rights and can act as if they claimed the plot in the first place. Owners cannot be removed from plots.

List Owners

/list-owners [plot name]
/owners [plot name]
Lists the owners of a given plot. If no plot is given then it lists the owners of the plot that you are currently standing in.
